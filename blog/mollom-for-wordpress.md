# Mollom for WordPress

[Matthias Vandermaesen](http://www.netsensei.nl) released [a Mollom plugin for Wordpress](https://wordpress.org/plugins/wp-mollom/). You can [read more about it on his blog](https://www.netsensei.nl/archives/tag/mollom/) or you can [download WP-Mollom from the Wordpress.org plugin directory](https://wordpress.org/plugins/wp-mollom/). *Thanks for all the hard work, Matthias!*
