# PHP5 class for Mollom

[Tijs Verkoyen](http://blog.verkoyen.eu/) of [CR Solutions](http://crsolutions.be) used the [Mollom API documentation](https://www.mollom.com/api) to create a [PHP5 class for Mollom](http://mollom.crsolutions.be/). Tijs' class is available under terms of [a BSD license](https://en.wikipedia.org/wiki/BSD_License), so you can use it to integrate [Mollom](https://mollom.com) into your PHP applications. We've added the class on the [Mollom download page](https://www.mollom.com/download). *Thanks Tijs!*
